
#  Pooling Pattern
**Analytical:** Centralised <br/>
**Data Movement**: Row level data <br/>
**Required Trust between TREs**: Very High <br/>

An emerging pattern from the DARE UK TREvolution programme is that a project specific network is created between the TREs, to either move or provide the researcher with a view of the data that makes it appear as if the data is all in one location. This approach is by far the most powerful for a user and the analyses and learning are almost unlimited.

This approach does require a very high level of trust between TREs as it would require effective row level sharing of data between TREs. Disclosure control would still be applied before the data left the TRE network.
