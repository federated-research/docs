# Shared Pattern
**Analytical:** Connected <br/>
**Data Movement**: Model Parameters <br/>
**Required Trust between TREs**: High <br/>

The most commons method to undertake federated learning is for the model parameters to be passed via a central node, or directly between the TREs. The key difference for a TRE is that results are combined and shared between other TREs, therefore there is a higher trust required between the TREs, as they must trust the sharing of model parameters with the researcher and the TREs.
