# Weaves

A **Weave** is a combination of a platform and computational pattern, which are linked by what disclosure control is allowed for that weave.

## Weave Identity

A weave must contain a single platform pattern and a single Computational Pattern. A single platform may have multiple weaves if it implements more than a single Computational Pattern.
