# Get an Access Token

An Access Token is required for a researcher to submit analysis to the Submission Layer.

<Steps>

#### Log in to the Submission Layer user interface.

For further detail read the [components guide](/five_safes_tes/quickstart/stackinfo#submission-layer).

#### Retrieve a token

From the Submission Layer navigate to **_API Access Token -> Copy to Clipboard_**

<Callout type="info">
  The token will expire after the `Expires` period which is shown under the
  title `Current Access Token`.
</Callout>

</Steps>
