# Adding a New Researcher to the Submission Layer

## Prerequisites

To add a researcher to the platform you need to:
- Create a user account in the Keycloak user management system (`Dare-Control` realm),
- Add the user to a Project in the project's `People` tab in the Submission Layer.
The user's records across Keycloak and the Submission Layer are linked by their `username`.

## Process

<Steps>
### Submission Manager logs onto the [Submission Layer UI](/five_safes_tes/reference_implementation/core_components#submission-ui)

### Open the 'People' page.

Select the 'People' tab at the top of the home page.
This will display the list of existing users.

<img src="/images/TREFX_PeoplePage.png" width="500"></img>

### Open the 'Create User' page.

Click the blue `+ Create New User` button at the top-right of the 'People' page.
This will open the 'Create User' page.

<img src="/images/TREFX_CreateUserPage.png" width="500"></img>

### Enter User details.

Enter the user details:

1. Full Name (mandatory)
2. Username (mandatory)
3. Email (mandatory)

<Callout>
  The username **must** match the username entered into Keycloak for the same
  user which was created in the
  [Adding a new user to KeyCloak](/submission/guides/keycloakAdminAddingNewUser)
  guide. This has to be checked manually.
</Callout>

### Submit information.

Once all details have been added click the blue 'Submit' button.
This will return you to the People page, where the new user will be displayed.

<img src="/images/TREFX_PersonPage_WithNewPerson.png" width="500"></img>

</Steps>
