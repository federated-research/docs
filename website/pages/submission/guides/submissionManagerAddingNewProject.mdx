# Adding a New Project to the Submission Layer

## Prerequisites and Notes

Required information:
- Project Name
- Start and End dates
- Project embargo status
- Researcher usernames to be associated with project

Optional (for this system) information:
- Project Owner, and contact
- Project Description

## Process

<Steps>
### Submission Manager logs onto the [Submission Layer](/five_safes_tes/quickstart/stackinfo#submission-layer)

### Open Projects page, and select Create Project

Select the 'Projects' tab at the top of the page,
and then click the blue '+ Create Project' button
on the top right of the Projects page.

<img src="/images/TREFX_ProjectPage.png" width="500"></img>

### Enter Project Information

The project information should be entered.
This includes:
1. Project Name (mandatory)
2. Start and End dates (mandatory)
3. Project Owner (optional)
4. Project Contact (email) (optional)
5. Mark if project is embargoed
6. Project Description (optional)
7. Display on UI (not currently modifiable)

<img src="/images/TREFX_CreateProjectPage.png" width="500"></img>

Once this information is entered press the blue 'Submit' button.
The new project will be visible on the Projects page.

<img src="/images/TREFX_ProjectPage_NewProject.png" width="500"></img>

### Add People to the Project

Click on the new project's name to open the project page for this.
Then select the 'People' tab within the project page,
and click the blue '+ Add User' button.

<img src="/images/TREFX_ProjectAddUser.png" width="500"></img>

This opens a menu containing the available users.
Select those that you wish to add to the project,
and then click the blue 'Add' button.

### Add TRE(s) to the Project

Select the 'TREs' tab within the project page,
and click the blue '+ Add TRE' button.
This opens a menu containing the available TREs.


Select those that you wish to add to the project,
and then click the blue 'Add' button.

<img src="/images/TREFX_ProjectAddTRE.png" width="500"></img>

### Request Approval from TRE

The 'TRE' tab for the project will now show that
the project status with the TRE is 'Undecided'.

<img src="/images/TREFX_ProjectAddTRE-Undecided.png" width="500"></img>

[Approval for the project](/activities/TREManagerApprovingProject)
should be requested from the TRE Manager.

<Callout>
The process for requesting approval is out of scope of this document.
</Callout>

### Approval Received from TRE

Once the TRE Manager has approved the project
then the TRE status will change to 'Approved'.

<img src="/images/TREFX_ProjectAddTRE-Approved.png" width="500"></img>


</Steps>
